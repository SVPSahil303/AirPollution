About This Project

AQI Intelligence is a Machine Learning powered platform designed to predict Air Quality Index (AQI) using environmental pollutant data and visualize pollution levels interactively.

Project Overview

This system analyzes pollution parameters such as PM2.5, PM10, NO₂ and O₃ to estimate the Air Quality Index using a trained Machine Learning model. The platform also provides interactive visualization tools including maps, charts and WHO guideline comparisons to help users understand air pollution levels in different locations.

Key Features

  • Machine Learning based AQI prediction
  • Live air pollution data from APIs
  • Interactive geospatial visualization
  • WHO pollution guideline comparison
  • Pollution distribution charts
  • Location based AQI prediction
